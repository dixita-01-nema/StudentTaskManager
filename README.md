# Student Task Manager

## Introduction
Student Task Manager is a Java project that helps students manage assignments, deadlines, and study tasks.

## Problem Statement
Students often forget important assignments and deadlines. This project helps them organize their work.

## Features
- Add tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks in a file
- Load tasks automatically when program starts

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- File Handling

## Project Structure
StudentTaskManager/
│── Task.java
│── TaskManager.java
│── Main.java
│── tasks.txt
│── README.md

## How to Run
1. Open terminal
2. Compile:
   javac *.java
3. Run:
   java Main
